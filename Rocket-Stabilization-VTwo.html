Rocket Stabilization V.2

 June 2023: Guidance system for a CO2-propelled rocket.  Four flight-control surfaces and rocket body, described in another linked project, were designed using Fusion 360 and printed using PLA with a FDM Printer.  Input for the guidance system for this rocket assembly is a 6 degree-of-freedom gyrometer.  These inputs were fed to an Arduino-based microcontroller, whose outputs controlled the flight-control surface via four servomotors.  Version 2.0 refers to the internal mounting of all guidance system components.
